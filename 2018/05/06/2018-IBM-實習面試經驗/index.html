2018 IBM 實習面試經驗

  3. 5/1 收到面試通知：
  4. 5/5 面試：

4/15 前投履歷：

我投了三個職位：

  1. Security Software Engineer - Front-End
  2. IT Specialist – Business Analyst / System Analyst
  3. Application Developer

4/17 IPAT：

那在 4/17 收到了 Email 被通知去做 IPAT 測驗

這個測驗分成兩部分：

  1. 數列找規律
  2. 英文版的國中高數學題

第一個部分我真的覺得很難，除了那種很簡單的平方或是不複雜的加減乘除有把握以外
其他幾乎都用猜的 XD

第二個部分題目本身不難，但是必須要在 2:15 內看懂題目並完成選擇
我英文的閱讀速度實在是太慢了，很多簡單題也因此沒做出來
題目大概問說 給你 2% 和 20% 的容液，分別需要多少才能配出 12% 的容液
大部分就是列出兩個式子找解就有的題目

5/1 收到面試通知：

因為我覺得我 IPAT 的測驗實在做太爛了
原本已經忘記有這回事，突然這天早上就收到 5/5 去 IBM 總部面試的通知
也是這個時候才知道我被分到 Application Developer

5/5 面試：

面試分成蠻多梯次，我這個職位至少有兩梯次以上
我是分到早上 10 點的梯次，一進大樓就看到全部穿著西裝的面試者
心想自己只穿襯衫是不是弱掉了XD

check-in 時，他會給你一張安全識別貼紙以及面試編號
編號分成 A~G 依職位分配

這時有點像營隊當小隊員排隊空閒的時間，所以有些人就開始聊了起來，還蠻好玩的XD

之後就是被帶到一個小圓桌的會議室團體面試
一開始是介紹這個職位的內容，然後問我們有沒有什麼問題
問完之後，每個人依編號介紹自己的名字（我很意外居然沒有自我介紹）
之後就被分配到一張白紙、兩篇英文文章及一個問題
有 3 分鐘的時間閱讀以及用白紙寫問題的答案（任何形式）
結束後有 1 分鐘的時間，抽籤決定表達自己的觀點

文章為智能建築的文章，問題為智能建築的資訊如何取得，相關的技術有哪些

表達完自己觀點後，分兩邊組隊
有 6 分鍾的時間討論，要整理我們這組的想法並有 2 分鐘報告

報告完後，8 分鐘的交叉提問
每個人有 1 分鐘的時間回答對面的問題及問對面問題（利用剛剛的報告）
最後每組花 1 分鐘做總結

之後每個人抽白紙決定題目（總共有五題）
我被分到的問題是
除了學校上課以外，在過去一年間還做了什麼事情增進能力或技術？
還有其他問題像是
Teamwork 經驗，舉實例
其他我忘了都是些常見的面試題不算困難
給你 2 分鐘寫 1 分鐘回答

這裡有很多只有 1 分鐘的時間，所以只能講重點
蠻多人有講超過 1 分鐘只好被要求停下來

最後是用人主管跟後面五位有點像工程師或 PM 的人問問題
他們會根據我們的履歷分別問一個或一些人問題

像是我被三個人問到說：

  1. 進實習後有什麼規劃
  2. Java 和 SQL 或是其他熟悉的語言有用來做過什麼
  3. 有在課堂上做過什麼 hands-on 的作業嗎

最後填個問卷，對於活動的建議之類的
就結束了

5/6 錄取

原本說 5/15 前會用 email 通知
結果隔天早上就接到電話了

被分到的部門是 iX 據說跟銀行業有關的工作
