Answer

Description

很多隻不同高度的河馬要走過固定高度的門
兩隻河馬可以背著使用Td的時間走 兩隻加起來的高度必須小於門的高度
一隻河馬走的話使用Ta的時間
找出最小通過門的時間

Input

C: 測資數量
N: 河馬數量
H: 門的高度
Ta: 單獨河馬經過門的時間
Td: 兩隻河馬背著走的時間

每筆測資：
N H Ta Td
接下來為N個河馬的高度輸入

Output

‘Case X: 時間’

Sample Input

Sample Output

Idea

最高的河馬配最矮的河馬 若還是不能通過門則高河馬必須自己走

Trap

Ta 可為0
Ta + Ta <= Td is fine.
